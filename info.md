# Abode Security

Custom Home Assistant integration for Abode Security Systems with enhanced features and control.

## Key Features

- **Complete System Control** - Monitor and control your Abode security system
- **Manual Alarm Trigger** - Trigger alarms with optional codes
- **Test Mode Support** - Enable test mode to prevent police dispatch
- **Multi-Platform Support** - Full support for all Abode device types
  - Security Panels
  - Sensors (door, window, motion)
  - Cameras
  - Locks
  - Lights
  - Covers
  - And more

## What's Included

This integration provides:
- Alarm control panel with full state management
- Service calls for alarm operations
- Switch controls for manual alarms and test mode
- Binary sensors for all door/window and motion sensors
- Camera feeds for video monitoring
- Lock and light controls
- Temperature and other environmental sensors

## Getting Started

1. Install via HACS
2. Add your Abode credentials
3. All your Abode devices will be automatically discovered

## Requirements

- Home Assistant 2024.1 or later
- Active Abode account
- Python 3.11+

See README.md for detailed setup and service documentation.
